# 🏡 SeokBang

🔔 **삼성 청년 SW 아카데미(SSAFY)** 1학기 관통 프로젝트

<br><br>

## **개요**

- 공공데이터를 활용하여 지역별, 아파트별 실거래가 조회 결과를 제공
- 지역별 게시판을 통해 유저 간의 상호 커뮤니케이션 공간 제공

<br><br>

## 프로젝트 기간

- 2022년 11월 16일 ~ 2021년 11월 25일
